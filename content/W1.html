W1

建立倉儲
先下載 portablekmol_base.7z    (Python395.7z     portablegit_2.31.1.7z)放在data下

安裝倉儲子模組

進入mdecourse/cad2021_hw點選use this template然後在自己的github帳號下幫倉儲命名

點選上方齒輪 settings 再點選 pages

更改 branch 

選 main 然後 save

然後將ipv4和ipv6拉到scite 將ipv6複製到ipv4然後再ipv4的scite裡把有ipv6的字言改為ipv4

接著開啟ipv4 git clone 自己網址.git 到自己隨身碟裡

再 stop 在打開 start_ipv4

輸入 pip install pyopenssl 安裝 pyopenssl 模組

再來就可以試試看 cms

有跑出網址就代表安裝成功

--------------------------------------------------------------------------------

接著告訴系統 user.name 跟 user.email

git config --global user.name 你的名字 (ex:40923111)

git config --global user.email 你的學校gm帳號 (ex:40923111@gm.nfu.edu.tw)

--------------------------------------------------------------------------------

設定 token 

按自己帳號下的setting

按 Developer settings

按 Personal access tokens>Generate new token

輸入密碼

Note打倉儲名稱將public_repo選起來

確認後將亂碼複製起來

打開隨身碟裡倉儲下的 .git 的 config

亂碼後加@就完成設定了

之後試試acp看看能不能上傳
